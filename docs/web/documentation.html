Program structure

3Depict has documentation regarding the program structure available. This is automatically generated using the Doxygen "Doxyfile" in the source tarball. The documentation provides data such as call and inheritance diagrams, hyperlinked function declarations and descriptions, and provides a useful "first glance" reference for working with the code. For a full program overview, try looking at the MainWindowFrame class.
